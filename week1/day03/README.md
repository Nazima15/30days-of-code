## 📌 문제: 백준 1003 - 피보나치 함수
https://www.acmicpc.net/problem/1003

### 🔎 문제 설명
피보나치 함수를 재귀적으로 호출할 때, 0과 1이 각각 몇 번 출력되는지 구하는 문제입니다.

### 💡 풀이 아이디어
- 단순 재귀로 풀면 시간 초과가 발생합니다.
- DP(동적 계획법) 또는 메모이제이션을 이용해 `0`과 `1`이 출력되는 횟수를 미리 구합니다.
- `dp[n][0]` = n일 때 0 출력 횟수  
- `dp[n][1]` = n일 때 1 출력 횟수  

### ✅ 예시
입력:
```

3
0
1
3

```

출력:
```

1 0
0 1
1 2

````
